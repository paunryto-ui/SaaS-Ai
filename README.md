## Tentang Proyek

**SaaS AI** adalah platform asisten kecerdasan buatan berbasis web yang dirancang agar siapa pun bisa berinteraksi dengan AI modern secara mudah dan nyaman. Tampilannya bersih dan intuitif, terinspirasi dari ChatGPT, namun dibangun sendiri dari nol oleh developer lokal Indonesia.

Proyek ini lahir dari semangat inovasi teknologi dalam negeri — dibuat sepenuhnya oleh **Wira**, seorang pengembang muda asal Indonesia yang percaya bahwa teknologi AI harus bisa diakses semua kalangan, tanpa batas.

## Fitur

- 💬 **Chat AI Real-time** — Percakapan cerdas yang responsif dengan model terbaik via OpenRouter
- 🌙☀️ **Dark & Light Mode** — Ganti tema sesuai preferensi dengan satu klik, tersimpan otomatis
- 🖼️ **Dukungan Gambar** — Upload dan analisis gambar langsung dalam percakapan
- 📋 **Riwayat Chat** — Semua sesi tersimpan selama browser aktif, bisa diakses lewat sidebar
- ⚡ **Tools Menu Pintar** — Akses cepat pencarian web, generate kode, terjemahan, rangkum teks
- 📱 **Responsif** — Tampilan optimal di semua ukuran layar, dari HP hingga desktop
- 🔒 **API Key Tersembunyi** — Kunci API terenkripsi, tidak pernah terekspos ke pengguna
- 🇮🇩 **Karya Indonesia** — Dibangun 100% oleh developer lokal dengan kebanggaan Indonesia

## Cara Pakai

1. Download atau clone repo ini
2. Buka file `saas-ai.html` di browser (Chrome, Edge, Firefox, Safari)
3. Langsung ketik pesan dan tekan **Enter** untuk mulai chat
4. Klik tombol **+** untuk akses tools tambahan (upload gambar, pencarian web, dll)
5. Klik ikon ☀️/🌙 di pojok kanan atas untuk ganti tema
6. Sidebar kiri untuk lihat riwayat chat dan mulai percakapan baru

> Tidak perlu install apapun. Cukup buka di browser, langsung jalan.
